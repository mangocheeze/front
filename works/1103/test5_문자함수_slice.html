    0123456789012345678901234567890
    ABCDEFGHIJKLMNOPQRSTUVWXYZABCopq
                    0987654321

length : 객체(문자열)의 길이

slice(a,b) : 지정된 문자열(a에서 b앞까지)을 추출해준다.

substring(a,b) : 지정된 문자열(a에서 b앞까지) 을 추출해준다

substr() :

replace(a,b) : a를 b로 치환

replaceAll(a,b) : a를 b로 모두 치환

toUpperCase() : 대문자로 치환

toLowerCase() : 소문자로 치환

concat(a,b) : a와 b를 결합한다

trim() : 양쪽 공백을 삭제한다

trimStart() : 앞쪽 공백만을 삭제한다

trimEnd() : 뒤쪽 공백만을 삭제한다

replaceAll() : 모든공백을 삭제한다

charCodeAt() : ()인덱스번째의 아스키코드출력

split() : 구분자를 기준으로 잘라준다.잘라진 각각의 요소는 배열로 저장된다

indexOf : 특정문자의 위치값을 찾아준다(찾는값이 없으면 -1출력)

lastIndexOf :특정문자의 위치를 뒤에서부터 찾아서 위치값을 찾아준다

search() : 특정문자의 위치값을 찾아준다(찾는값이 없으면 -1출력)

match() : 찾는 문자열이 있으면 '문자열'을 반환, 없으면 '공백'을반환

includes() : 찾는 문자열을 포함하고있으면 true,없으면 false를 반환한다
